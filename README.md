# PHBS QPS 2024

本仓库包含工具和脚本，用于获取美国消费者价格指数 (CPI) 数据并计算最近四个季度的通胀率。

## 仓库结构
- `data`：用于存储原始和处理后的数据。
- `notebooks`：用于分析和可视化的 Jupyter 笔记本。
- `scripts`：用于获取和处理数据的 Python 脚本。
- `src`：可重用的辅助模块。
